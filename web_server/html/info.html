基础攻击教程（来自译者）

在“热点”页面中扫描，并选择需要攻击的热点。
在“攻击”页面中找到Deauth一行，点击START开始攻击。
*开始攻击后，所有通过与攻击相同信号频率的无线连接至该热点的设备瞬间全部断网并无法进行再次连接，关闭后恢复。
